Aberrant Barter

Aberrant Barter is an app where users can create a profile, and post items that they would be willing to trade. Other users will then post an offer with a different item that they want to trade for the item you have posted. Users are able to accept an offer that a user has posted and close out the trade. Creative and unusual trades is the premise of the app, so get Aberrant and start making some trades!

Let's Fly

Need to book a flight? Use Let's Fly!

This easy to use app allows travelers to book flights on the fly!

Feeling spontaneous? Lets Fly can also choose a random destination for you, and if it's not to your liking feel free to spin again for a new destination.

This was a group project a team and I built to solve the problem of, "Where should I travel to?". I was responsible for keeping the team on task as the scrum master. I also helped the team with the front-end development, as well as some of the back-end.

Login using a Gmail account in order to view save function.
